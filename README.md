# Linguagem-Esoterica Sertaocode
SertãoCode 🌵💻 é uma linguagem esotérica criada para fins pedagógicos e divertidos em uma disciplina da faculdade. Inspirada em O Auto da Compadecida, usa jargões e falas icônicas para representar comandos, tornando o aprendizado de programação mais leve, cultural e criativo.
